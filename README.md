# CellPeek

CellPeek 是一个用于飞书 / Lark 表格的 Chrome MV3 插件。点击单元格后，会在浏览器右侧侧栏展示单元格内容，并支持 `自动 / MD / JSON / 原文` 视图切换。

## 功能

- 点击插件按钮后，在浏览器右侧打开侧栏
- 点击单元格后自动读取并展示内容
- 展示内容可选中复制，链接可点击
- 支持单栏、左右分栏、上下分栏
- 分栏模式支持拖拽调节比例
- 每个分栏支持锁定 / 解锁

## 安装

1. 打开 `chrome://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择目录 `e:\chrome插件`
5. 刷新飞书表格页面后使用

## 使用

1. 在飞书表格页面点击浏览器扩展图标
2. 右侧会打开 CellPeek 侧栏
3. 点击任意单元格，即可在侧栏查看内容

按钮说明：
- `单`：单栏
- `左右`：左右分栏
- `上下`：上下分栏
- `🔓 / 🔒`：解锁 / 锁定当前分栏
- `自动 / MD / JSON / 原文`：切换当前分栏展示模式

## 生效范围

插件仅在以下域名生效：
- `feishu.cn`
- `larksuite.com`
- `larkoffice.com`
